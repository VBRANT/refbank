RefBank Search Portal Help

The RefBank search portal offers the following functionality:
  • Search the Bibliography of Life via the search form:
    • Full text field: full text search in plain reference strings
    • Type selector: filter by type of referenced work
    • Author(s) field: search by author names
    • Title field: search by publication titles or parts thereof
    • Year field: search by year of publication
    • Origin (Journal/Publisher) field: search by journal name and volume, publisher and location, or book title
  • Double-click a reference string in the search result to select it for copy & paste
  • Hover a reference string in the search result with the mouse to reveal further options:
    • Retrieve references in multiple file formats for import in bibliography management software
    • Retrieve references in multiple styles, e.g. for immediate use in a publication
    • Contribute to the Bibliography of Life:
      • Parse Reference / Refine Parsed Reference: Open a reference string in a working dialog to mark individual attributes
      • Edit Reference: Edit a reference string, e.g. to correct a typo (the original one becomes a duplicate of the edited version)
      • Delete: Flag a reference string as deleted, so it does not show in search results by default (the Show references flagged as deleted button makes these references show again, and you can un-delete them)
    • Show Duplicates: show a list of duplicates of a reference:
      • Make Representative: make a reference the representative of the cluster of duplicates it belongs to, so it shows up in search results instead of the current one
      • Not a Duplicate: remove a reference from a cluster of duplicates, putting it back on the top level, so it shows up in search results by itself
      • Drag reference A and drop it on reference B to indicate that A is a duplicate of B
